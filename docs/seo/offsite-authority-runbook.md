# Off-Site Authority Runbook (Exchange-First)

This runbook supports the 90-day target of 9-12 relevant referring domains.

## Qualification Rules
- Prioritize relevant domains: student housing, university resources, relocation blogs, international student portals.
- Reject paid link schemes, private blog networks, and low-quality directory spam.
- Require crawlable, indexable pages for accepted placements.

## Monthly Cadence
1. Build outreach list (20-30 targets/month).
2. Pitch one value asset (guide, checklist, or local housing explainer).
3. Secure 3-4 relevant mentions/month.
4. Verify indexing and referral quality.

## Target Source Types
- WHU exchange/community resource pages
- Student accommodation directories with editorial standards
- Expat/student relocation publications
- Local Vallendar/Koblenz community resources

## Quality Scoring
- High: contextually relevant, editorial mention, indexable page, referral traffic.
- Medium: relevant directory/profile with clear user value.
- Low: weak context, thin page, or non-indexed placement.

## Monthly Deliverables
- Outreach list CSV
- Outreach copy pack (DE/EN)
- Placement tracker update
- Summary in weekly KPI report
